# **App Name**: LostIt

## Core Features:

- User Authentication: Secure user authentication via Firebase with email/password and university affiliation during signup.
- Report Lost Item: Submission form for users to report lost items, including title, detailed description, category, optional photo, contact information, University and location. Location input via free-form text field.
- Report Found Item: Submission form for users to report found items, including title, description, category, optional photo, contact information, University, and location. Location input via free-form text field.
- University-Specific Item Display: Display of item cards filtered by the user's registered university with a detailed view option and owner-only "Reached Owner" status update.
- Tabbed Item Filtering: Tabbed browsing UI for quickly filtering display of lost vs found items.

## Style Guidelines:

- Primary color: A warm yellow (#FDD835), reminiscent of campus lighting, and implying safety.
- Background color: Light gray (#FAFAFA), offering a clean and neutral backdrop that highlights the app's content.
- Accent color: A muted orange (#FFAB40), providing contrast to call attention to actionable items like CTAs and alerts.
- Body and headline font: 'PT Sans' (sans-serif), balancing modern aesthetics with approachability and readability.
- Simple and universally recognized icons for categories (keys, books, electronics etc.)
- Clean, card-based layout with clear sections for different item attributes. Prioritize clarity for easy scanning and comprehension.
- Subtle transition animations when filtering/sorting items or marking them as 'Reached Owner'.
